【对象增强的三种方式】
1.继承
2.装饰模式
3.代理模式


【装饰模式的优缺点】
优点：
 1.装饰类和被装饰类是可以独立的，低耦合的。互相都不用知道对方的存在
 2.装饰模式是继承的一种替代方案，无论包装多少层，返回的对象都是is-a的关系(上面的例子：包装完还是Phone类型)。
 3.实现动态扩展，只要继承了装饰器就可以动态扩展想要的功能了。
缺点：
 1.多层装饰是比较复杂的，提高了系统的复杂度。不利于我们调试~
